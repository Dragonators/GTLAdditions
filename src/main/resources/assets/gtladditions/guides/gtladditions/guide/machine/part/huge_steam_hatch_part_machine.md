# Huge Steam Input Hatch

<Column gap="15" fullWidth={true}>

<BlockImage id="gtladditions:huge_steam_input_hatch" scale="8" />

<Column gap="2" fullWidth={true}>

* You can craft it in the <Color color="#AA00AA">**EV**</Color> stage.
* Huge Steam Input Hatch can only be installed on steam multiblock structures, replacing a Steam Input Hatch or Large Steam Input Hatch.
* It raises the processing recipe voltage tier of steam multiblocks to <Color color="#FFAA00">**HV**</Color>.
* Its fluid capacity is 2147483647 mB and it only accepts steam.
* It can reduce all processing recipe duration to 1t. Its steam-to-EU conversion ratio is:

<Latex math = "125\ \mathrm{mB/t} = 1\ \mathrm{EU/t}" />

</Column>

<Column gap="2" fullWidth={true}>

> Watch your steam supply. The Large Steam Macerator can gain byproducts through this.
