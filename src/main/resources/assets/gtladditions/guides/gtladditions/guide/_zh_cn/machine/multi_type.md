# 跨配方模式并行

<Column gap="15" fullWidth={true}>

<Column gap="2" fullWidth={true}>

跨配方模式并行是一类机器能力：当一台机器支持多个配方类别时，它可以在同一轮工作中同时处理不同类别下的配方，而不是只围绕单一配方类别运行。

</Column>

<Column gap="2" fullWidth={true}>

* 这类能力与普通并行不同：普通并行主要放大单个配方的执行规模，跨配方模式并行则允许多个配方类别共同参与同一轮工作。
* <ItemLink id="gtladditions:forge_of_the_antichrist" /> 可以在超维度熔炼、恒星热能熔炼、终极物质锻造之间进行跨配方并行。
* <ItemLink id="gtladditions:apocalyptic_torsion_quantum_matrix" /> 可以在量子操纵者、深度化学扭曲仪、奇点压缩之间进行跨配方并行。
* 实际能同时推进多少配方，仍会受到机器线程、并行能力、输入与能源供应的限制。

</Column>

</Column>
