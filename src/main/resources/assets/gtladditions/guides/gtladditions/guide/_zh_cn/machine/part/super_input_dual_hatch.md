# 超级输入总成

<Column gap="15" fullWidth={true}>

<BlockImage id="gtladditions:super_input_dual_hatch" scale="8" />

<Column gap="2" fullWidth={true}>

* 你需要在 <Color color="#00AA00">**UIV**</Color> 阶段才能制作它。
* 超级输入总成拥有大量输入槽位，可以容纳比巨型输入总成更多的物品和流体。
* 它拥有 37 个物品槽，每个物品槽可以容纳 2,147,483,647 (2.14G) 个物品。
* 它拥有 24 个流体槽，每个流体槽容量为 2,251,799,813,685,247 (2.2TB) mB。

</Column>

<Column gap="2" fullWidth={true}>

> 返回物品或流体时，单次转移仍受相邻目标能力和接口限制影响。如果你在仓内输入了大量流体，可能需要多次返回或借助外部抽取 (上限2.14MB)。
