# Molecular Assembler and Infinity Crafting Module

<Column gap="15" fullWidth={true}>

<Row>
    <BlockImage id="gtceu:molecular_assembler_matrix" scale="4" />

    <BlockImage id="gtladditions:dimension_focus_infinity_crafting_array" scale="4" />

<Column gap="2" fullWidth={true}>

* <ItemLink id="gtceu:molecular_assembler_matrix" /> can work with <ItemLink id="gtladditions:dimension_focus_infinity_crafting_array" />.
* Installing that module unlocks Infinity Crafting mode for the Molecular Assembler.
* In Infinity Crafting mode, Molecular Assembler processing time is fixed to 1 tick and has infinite parallelism.
* If the module becomes invalid or disconnected, the Molecular Assembler returns to normal mode.

</Column>

For the module itself, see [Dimension Focus Infinity Crafting Array](../machine/controller/dimension_focus_infinity_crafting_array.md).
