# 分子操纵者与无尽合成模块

<Column gap="15" fullWidth={true}>

<Row>
    <BlockImage id="gtceu:molecular_assembler_matrix" scale="4" />

    <BlockImage id="gtladditions:dimension_focus_infinity_crafting_array" scale="4" />

<Column gap="2" fullWidth={true}>

* <ItemLink id="gtceu:molecular_assembler_matrix" /> 可以与 <ItemLink id="gtladditions:dimension_focus_infinity_crafting_array" /> 联动。
* 安装该模块后，分子操纵者会解锁无尽合成模式。
* 无尽合成模式下，分子操纵者的处理时间固定为 1 tick，并拥有无限并行。
* 模块失效或断开后，分子操纵者会回到普通模式。
