# Heart of the Universe

<BlockImage id="gtladditions:heart_of_the_universe" scale="8"/>

<Column gap="20" fullWidth={true}>

<Column gap="2" fullWidth={true}>

* The final generator. It uses <ItemLink id="gtladditions:strange_annihilation_fuel_rod" /> to annihilate planes into energy in the wireless energy network.
* The structure requires <ItemLink id="gtladditions:wireless_energy_network_output_terminal" />.
* Each plane is annihilated within 1 second and outputs power equivalent to 4096 <ItemLink id="gtmthings:creative_energy_hatch" /> into the wireless energy network.

</Column>

<Column gap="2" fullWidth={true}>

### Plane Unfolding

* <ItemLink id="gtladditions:heart_of_the_universe" /> processes one plane at a time by default.
* The structure can install <ItemLink id="gtladditions:thread_modifier_hatch" />.
* After installation, it can unfold more planes and process them at the same time.
* The number of planes that can be processed simultaneously depends on the threads provided by <ItemLink id="gtladditions:thread_modifier_hatch" />.

</Column>
