# Nexus Satellite Factory

<Column gap="20" fullWidth={true}>

<Column gap="2" fullWidth={true}>

* A <Color color="#FFFF00">**UXV**</Color> stage module series for <ItemLink id="gtladditions:subspace_corridor_hub_industrial_array" />.
* Nexus Satellite Factory must be installed on <ItemLink id="gtladditions:subspace_corridor_hub_industrial_array" /> to run. It will not execute recipes when disconnected from the hub.
* If the hub is not formed, not powered, or not kept in a valid working state, connected modules also stop working.
* One <ItemLink id="gtladditions:subspace_corridor_hub_industrial_array" /> can connect at most 138 Nexus Satellite Factory modules.

</Column>

<Column gap="2" fullWidth={true}>

### Module Tiers

* <ItemLink id="gtladditions:nexus_satellite_factory_mk1" />: for Lathe, Bender, Compressor, Forge Hammer, Cutter, Extruder, Mixer, Wiremill, Forming Press, and Polarizer type processing.
* <ItemLink id="gtladditions:nexus_satellite_factory_mk2" />: for Rock Breaker, Ore Washer, Centrifuge, Electrolyzer, Sifter, Macerator, Dehydrator, Thermal Centrifuge, and Electromagnetic Separator type processing.
* <ItemLink id="gtladditions:nexus_satellite_factory_mk3" />: for Evaporation, Autoclave, Extractor, Brewery, Fermenter, Distillation, Fluid Heater, Fluid Solidifier, and Chemical Bath type processing.
* <ItemLink id="gtladditions:nexus_satellite_factory_mk4" />: for Canner, Arc Furnace, Lightning Processor, Assembler, Precision Assembler, and Circuit Assembler type processing.

</Column>

<Column gap="2" fullWidth={true}>

### Parallel And Threads

* Nexus Satellite Factory supports laser power and can install <ItemLink id="gtladditions:thread_modifier_hatch" />.
* Under normal conditions, modules provide parallel capability based on their own tier, structure, and installed parts.
* When the hub embeds a full 512 <ItemLink id="gtladditions:astral_array" link={false} linkColor={true} /> and unlocks Paradoxical Attainment Theory, connected modules receive infinite parallel and infinite threads from the hub. One <ItemLink id="gtladditions:compressed_astral_array" link={false} linkColor={true} /> fills this cap.

</Column>

</Column>
