# Super Input Dual Hatch

<Column gap="15" fullWidth={true}>

<BlockImage id="gtladditions:super_input_dual_hatch" scale="8" />

<Column gap="2" fullWidth={true}>

* You need to reach the <Color color="#00AA00">**UIV**</Color> stage before you can craft it.
* Super Input Dual Hatch has many input slots and can hold more items and fluids than a Huge Input Dual Hatch.
* It has 37 item slots. Each item slot can hold 2,147,483,647 (2.14G) items.
* It has 24 fluid slots. Each fluid slot has a capacity of 2,251,799,813,685,247 (2.2TB) mB.

</Column>

<Column gap="2" fullWidth={true}>

> Returning items or fluids is still limited by the adjacent target capability and interface per transfer. If you put a very large amount of fluid into the hatch, you may need multiple return operations or external extraction (2.14MB limit).

~~You can also use primitive robot arms or primitive fluid regulators to extract quickly.~~

</Column>
