# Integrated Ore Processor Recipes

<Column gap="15" fullWidth={true}>

<Column gap="2" fullWidth={true}>

These recipes belong to the <ItemLink id="gtceu:integrated_ore_processor" /> recipe category. Actual output amounts depend on the local ore processing multiplier config.

</Column>

<Column gap="2" fullWidth={true}>

## Jasper Ore Processing

* The circuit 24 recipe processes jasper ore tag inputs.
* It uses distilled water as the fluid input, mainly outputs jasper dust, and includes chance outputs such as talc dust, boron dust, and raw tengam dust.
* This recipe supplements raw tengam acquisition; check JEI for exact output amounts.

</Column>

<Column gap="2" fullWidth={true}>

## Platinum Group Ore Processing

* Circuit 8 recipes process <ItemLink id="gtceu:cooperite_ore" />, <ItemLink id="gtceu:bornite_ore" />, <ItemLink id="gtceu:tetrahedrite_ore" />, and <ItemLink id="gtceu:chalcocite_ore" />, producing the corresponding purified crushed ore.
* Circuit 9 recipes use an additional matching fluid input to improve byproduct handling.
* <ItemLink id="gtceu:chalcocite_ore" /> only has the circuit 8 recipe, not the circuit 9 recipe.

</Column>
