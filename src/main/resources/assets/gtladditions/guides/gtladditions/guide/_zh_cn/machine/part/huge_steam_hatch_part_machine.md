# 巨型蒸汽输入仓

<Column gap="15" fullWidth={true}>

<BlockImage id="gtladditions:huge_steam_input_hatch" scale="8" />

<Column gap="2" fullWidth={true}>

* 你可以在 <Color color="#AA00AA">**EV**</Color> 阶段制作它。
* 巨型蒸汽输入仓只能安装在多方块蒸汽结构上，用来替代蒸汽输入仓或大型蒸汽输入仓。
* 它能将多方块蒸汽结构的处理配方电压等级提升到 <Color color="#FFAA00">**HV**</Color> 级别。
* 它的流体容量为 2147483647 mB，只接受蒸汽。
* 它可以将所有处理配方的耗时变成 1t，蒸汽转换 EU 的比率为：

<Latex math = "125\ \mathrm{mB/t} = 1\ \mathrm{EU/t}" />

</Column>

<Column gap="2" fullWidth={true}>

> 请注意你的蒸汽库存。大型蒸汽研磨机可以借此获得副产。

~~跟大型蒸汽输入仓一样，不要用在超维度蒸汽熔炉上。~~

</Column>

</Column>
