# Fuxi Bagua Heaven-forging Furnace

<BlockImage id = "gtladditions:fuxi_bagua_heaven_forging_furnace" scale = "8"/>

<Column gap="20" fullWidth={true}>

<Column gap="2" fullWidth={true}>

* Fuxi Bagua Heaven-forging Furnace integrates four advanced material processing lines.
* The machine supports parallel control hatches, laser hatches, cross-recipe parallelization, and coil tiers.
* The structure can install <ItemLink id="gtladditions:thread_modifier_hatch" />.

</Column>

<Column gap="2" fullWidth={true}>

### Recipe Modes

* Stellar Ignition: converts specified fluids into their matching plasmas.
* Chaotic Alchemy: merges some alloy smelting and cooling steps into shorter processes.
* Molecular Deconstruction: directly deconstructs dusts of fluid-capable materials into fluids.
* Ultimate Material Forge: processes Ultimate Material Forge recipes.
