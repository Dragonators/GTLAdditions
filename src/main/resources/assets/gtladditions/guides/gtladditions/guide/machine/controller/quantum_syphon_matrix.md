# Quantum Syphon Matrix

<BlockImage id="gtladditions:quantum_syphon_matrix" scale="8" />

<Column gap="15" fullWidth={true}>

<Column gap="2" fullWidth={true}>

* Quantum Syphon Matrix processes Voidflux Reaction recipes. It is a more efficient gas collector that consumes large amounts of power to obtain air, nether air, ender air, and their liquid versions.
* In the Barnarda C dimension, using <ItemLink id="gtladditions:barnarda_data" /> as input can obtain Barnarda Air.
* The structure can install <ItemLink id="gtladditions:thread_modifier_hatch" />.

</Column>
