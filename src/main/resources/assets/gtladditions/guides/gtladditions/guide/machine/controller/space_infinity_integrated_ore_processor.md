# Space Infinity Integrated Ore Processor

<BlockImage id="gtladditions:space_infinity_integrated_ore_processor" scale="6" />

* This is a MAX-stage machine that processes space ore recipes with infinite parallel and infinite threads.
* Before each work attempt, it consumes 100 B <FluidLink id="gtceu:stellar_energy_rocket_fuel" />.
* One <ItemLink id="gtladditions:spectral_analysis_hatch" /> can be installed. When Spectral Analysis Hatch is perfectly tuned, this machine doubles recipe output.
