# Infinity Input Dual Hatch

<Column gap="15" fullWidth={true}>

<BlockImage id="gtladditions:infinity_input_dual_hatch" scale="8" />

<Column gap="2" fullWidth={true}>

* You need to reach the late <Color color="#FFFF00">**UXV**</Color> stage before you can craft it.
* Infinity Input Dual Hatch is a late-game input dual hatch for multiblocks, providing infinite item input and fluid input capability.
* It aggregates stored contents by item or fluid type instead of using a fixed number of normal slots.
* Each item type count and each fluid type capacity can reach Long.MAX level.

</Column>

<Column gap="2" fullWidth={true}>

## ME Network Optimization

* Infinity Input Dual Hatch has special optimization for ME networks.
* Unlike <ItemLink id="gtladditions:super_input_dual_hatch" />, it can return all internal contents to the ME network at once, like an unmarked ME Interface.
* An AE Pattern Provider can input Long.MAX-level items and fluids into it at once without truncation.
