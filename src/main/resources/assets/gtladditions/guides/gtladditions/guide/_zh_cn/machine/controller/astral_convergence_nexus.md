# 星穹聚合枢纽

<BlockImage id="gtladditions:astral_convergence_nexus" scale="8"/>

<Column gap="15" fullWidth={true}>

<Column gap="2" fullWidth={true}>

* <ItemLink id="gtladditions:astral_convergence_nexus" /> 是 <ItemLink id="gtceu:space_elevator" /> 的模块机器，主要处理太空组装模块配方。
* 它需要连接到 <ItemLink id="gtceu:space_elevator" />；<ItemLink id="gtceu:space_elevator" /> 未满足运行条件时，模块不会继续推进配方。
* 机器支持激光输入和跨配方并行。
* 结构允许安装 <ItemLink id="gtladditions:thread_modifier_hatch" />。

</Column>

<Column gap="2" fullWidth={true}>

### 模块等级

* <ItemLink id="gtladditions:astral_convergence_nexus" /> 的最大并行由 <ItemLink id="gtceu:space_elevator" /> 中安装的 <ItemLink id="gtlcore:power_module_2" /> 至 <ItemLink id="gtlcore:power_module_7" /> 提供的等级决定。
* 模块等级越高，该模块可获得的并行上限越高。
* 成型后，界面会显示当前是否已连接 <ItemLink id="gtceu:space_elevator" />，以及可用的最大并行。
