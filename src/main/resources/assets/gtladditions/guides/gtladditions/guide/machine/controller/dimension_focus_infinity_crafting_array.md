# Dimension Focus Infinity Crafting Array

<BlockImage id="gtladditions:dimension_focus_infinity_crafting_array" scale="8"/>

<Column gap="20" fullWidth={true}>

<Column gap="2" fullWidth={true}>

* A simple automation solution for <ItemLink id="avaritia:extreme_crafting_table" />. Goodbye, packets.
* When running as an independent machine, its maximum parallel is 4096.
* The machine supports laser input and cross-recipe parallelization.
* The structure can install <ItemLink id="gtladditions:thread_modifier_hatch" />.

</Column>

<Column gap="2" fullWidth={true}>

### Infinity Crafting Mode

* When <ItemLink id="gtladditions:dimension_focus_infinity_crafting_array" /> is installed as a module on <ItemLink id="gtceu:molecular_assembler_matrix" />, it unlocks Infinity Crafting mode for that machine.
* In Infinity Crafting mode, <ItemLink id="gtceu:molecular_assembler_matrix" /> processing time is fixed to 1 tick and it has infinite parallel.
* If the module becomes invalid or disconnects, <ItemLink id="gtceu:molecular_assembler_matrix" /> returns to normal mode.
